# 🌀 Turtle Spirograph Generator

This project uses Python's built-in `turtle` graphics module to draw a colorful spirograph pattern. Each circle is drawn with a random RGB color, creating a vibrant and dynamic visual effect.

## 🐍 Requirements

- Python 3.x
- No external libraries required (uses `turtle` and `random`, which are part of Python's standard library)

## 🚀 How It Works

- The turtle draws circles in a loop, slightly rotating its heading each time.
- The rotation angle is determined by the `size` argument (e.g., `5` degrees).
- A random color is applied to each circle using `t.colormode(255)` and `random.randint`.

## 🧠 Learnings

This small project helps reinforce:
- Using the `turtle` graphics library
- Random color generation
- Basic geometry and angle rotation
- Function definition and loop control
